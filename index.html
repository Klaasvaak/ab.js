AB.js is an easy to use JavaScript based A/B testing framework.

Each variation is created by writing small JavaScript functions that modify the page.

When the page is loaded, AB.js selects one of your JavaScript variation functions at random, runs it and stores the chosen variation name in a cookie.

When the user visits the page again, the same variation is shown. This allows you to test per user rather than per page load.

AB.js then creates a Google analytics custom variable with the name of the variation. You can then compare the performance of each variation in the Demographics -> Custom Variables section of Google Analytics.

The framework is completely self contained and does not require any other libraries such as jQuery.

View AB.js on GitHub

Browser Support

AB.js should work in most modern browsers and it has been tested in the following:

  • Internet Explorer 6+
  • Firefox 4+
  • Chrome
  • Safari
  • Opera

Now for a working example with Pandas...

You can copy the below code into the head section of your page and modify it as needed.

<script type="text/javascript" src="http://aptonic.github.com/ab.js/ab.js"></script>

<script type="text/javascript">

var myAbTest = ABTest({
    name: "Example Test",
    customVarSlot: 1,
    variations: {
        first_variation: function () {
            document.getElementById("panda").src = "images/panda2.jpg";
            document.getElementById("footer-text").innerHTML = "This is the first variation.";
        },
        another_variation: function () {
            document.getElementById("panda").src = "images/panda3.jpg";
            document.getElementById("footer-text").innerHTML = "This is another variation.";
        },
        control: function () { /* Empty function. */
        }
    }
});

</script>
