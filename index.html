AB.js is an easy to use JavaScript based A/B testing framework.

Each variation is created by writing small JavaScript functions that modify the page.

When the page is loaded, AB.js selects one of your JavaScript variation functions at random, runs it and stores the variation in a cookie. This allows you to test per user rather than per page load.

AB.js then creates a Google analytics custom variable with the name of the variation. You can then compare the performance of each variation in the Demographics -> Custom Variables section of Google Analytics.

Here's a working example with Pandas

<script type="text/javascript">

var myAbTest = ABTest({
    name: "Example Test",
    customVarSlot: 1,
    variations: {
        first_variation: function () {
            document.getElementById("panda").src = "images/panda2.jpg";
            document.getElementById("footer-text").innerHTML = "This is the first variation.";
        },
        another_variation: function () {
            document.getElementById("panda").src = "images/panda3.jpg";
            document.getElementById("footer-text").innerHTML = "This is another variation.";
        },
        control: function () { /* Empty function. */
        }
    }
});

</script>
